# vue-paged-media

A Vue library for previewing paged media layouts before printing.

## Development

- Install dependencies:

```bash
vp install
```

- Run the unit tests:

```bash
vp test
```

- Start the documentation site:

```bash
vp run docs:dev
```

- Build the documentation site:

```bash
vp run docs:build
```

- Build the library:

```bash
vp pack
```
